9 """ASGI application middleware for Engine.IO.
10
11 This middleware dispatches traffic to an Engine.IO application. It can
12 also serve a list of static files to the client, or forward unrelated
13 HTTP traffic to another ASGI application.
14
15 :param engineio_server: The Engine.IO server. Must be an instance of the
16 ``engineio.AsyncServer`` class.
17 :param static_files: A dictionary with static file mapping rules. See the
18 documentation for details on this argument.
19 :param other_asgi_app: A separate ASGI app that receives all other traffic.
20 :param engineio_path: The endpoint where the Engine.IO application should
21 be installed. The default value is appropriate for
22 most cases. With a value of ``None``, all incoming
23 traffic is directed to the Engine.IO server, with the
24 assumption that routing, if necessary, is handled by
25 a different layer. When this option is set to
26 ``None``, ``static_files`` and ``other_asgi_app`` are
27 ignored.
28 :param on_startup: function to be called on application startup; can be
29 coroutine
30 :param on_shutdown: function to be called on application shutdown; can be
31 coroutine
32
33 Example usage::
34
35 import engineio
36 import uvicorn
37
38 eio = engineio.AsyncServer()
39 app = engineio.ASGIApp(eio, static_files={
40 '/': {'content_type': 'text/html', 'filename': 'index.html'},
41 '/index.html': {'content_type': 'text/html',
42 'filename': 'index.html'},
43 })
44 uvicorn.run(app, '127.0.0.1', 5000)
45 """
