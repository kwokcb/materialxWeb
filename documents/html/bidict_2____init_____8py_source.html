28"""The bidirectional mapping library for Python.
29
30----
31
32bidict by example:
33
34.. code-block:: python
35
36 >>> from bidict import bidict
37 >>> element_by_symbol = bidict({'H': 'hydrogen'})
38 >>> element_by_symbol['H']
39 'hydrogen'
40 >>> element_by_symbol.inverse['hydrogen']
41 'H'
42
